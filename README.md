# Video Scene Board

1本の動画を、シーンごとのカードで設計・管理し、さらにTODOタスクもステータスごとに管理できるアプリです。

## 概要
このアプリは以下を目的としています。

- 動画の構成（シーン）をカード形式で管理
- シーンの並び替え（ドラッグ＆ドロップ）
- 動画制作のTODOをカンバン形式で管理
- 複数動画を横断して管理

## 技術構成

- Frontend: React + Vite
- Backend: FastAPI
- Database: SQLite
- Drag & Drop: dnd-kit

## 主な機能

### 動画管理

- 動画の作成 / 編集 / 削除
- ステータス管理（draft / in_progress / done）
- コンセプト・ターゲット・ゴールの設定

### シーン管理

- シーンの作成 / 編集 / 削除
- ドラッグ＆ドロップによる並び替え
- 台本 / 素材 / キャラ / 秒数 / メモ管理

### TODO管理

- 未着手 / 作業中 / 完了
- 優先度（高 / 中 / 低）
- シーン紐付け
- ドラッグ＆ドロップでステータス変更
